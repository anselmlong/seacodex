Regional Codex Hackathon — Singapore (Sea x OpenAI)
Judging Process
Presentation format (both rounds): 6 min demo/pitch + 2 min Q&A. Cover problem, solution, and use of Codex. Judges score against shared criteria grounded in what you built and can demonstrate.
Round 1 — Preliminary: All teams present to a dedicated pair of Sea & OpenAI judges. Top 5 advance to finals.
Round 2 — Finals: Top 5 teams present on the main stage in front of all participants and a panel of final judges. Top 3 announced after this round.
Judging Criteria
Problem & Solution Fit — Is the problem real? Is the solution the right answer to it?
Build Quality — Does it work and hold up under use? The product is judged, not the pitch.
Insight & Originality — Is the approach genuinely novel, or just common industry practice?
Real-World Value — Based on the working demo: who benefits, and how significantly?
Alignment with Build Direction — Does the build meaningfully reflect at least one build direction?
Use of Codex — How deeply was Codex leveraged to build this product?
What to Submit
One submission per team via Google Form by 5:00 PM sharp. Late submissions are disqualified.
GitHub Repo — New public repository created for this hackathon.
Project Writeup — Written summary of the problem, solution, build direction it aligns with, and how Codex was utilised.
Demo Link — Live demo or prototype link (optional).
Declarations — State any existing OSS/personal project used, and what was built exclusively within this hackathon.
Hackathon Rules
All work must be done on-site. Teams may start from scratch or build on an existing personal or open-source project.
Building on an existing project is allowed only if:
New extensions are significant and substantial. Only work built during the hackathon will be judged.
Built in a new public repository.
Link to the existing repository/repositories provided in your README and submission form.
State clearly what was built during the hackathon in your README and submission form.
Any infringement of these rules may result in disqualification.
